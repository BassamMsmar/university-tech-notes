مقدمة في بناء واجهات المستخدم الأمامية (Front-End Development)

مقدمة في الويب

يعتبر الويب المنصة الأساسية في العصر الحديث لتقديم التطبيقات وتوصيل المعلومات. يتكون أي تطبيق ويب قياسي من جانبين: الواجهة الأمامية (Front-End) وهي ما يتفاعل معه المستخدم، والواجهة الخلفية (Back-End) المسؤولة عن معالجة البيانات.

يركز هذا المقرر حصراً على تقنيات الواجهة الأمامية والتي تشكل اللبنة الأساسية لأي مطور تطبيقات.

ملاحظة هامة: تعمل المتصفحات (مثل Google Chrome و Safari) كمترجمات لهذه التقنيات، حيث تستقبل الملفات من الخادم وتقوم بعرضها للمستخدم بشكل مرئي.

أساسيات هيكلة الصفحات (HTML)

لغة ترميز النص الفائق (Hypertext Markup Language) هي البنية التحتية لأي صفحة ويب. تستخدم HTML سلسلة من العناصر (Tags) لتغليف أجزاء مختلفة من المحتوى لجعلها تبدو أو تتصرف بطريقة معينة.

المكونات الأساسية

  • العناوين (Headings): تستخدم لعناوين الأقسام الرئيسية والفرعية، من h1 إلى h6.
  • الفقرات (Paragraphs): عنصر p مخصص للنصوص الطويلة.
  • الروابط (Links): عنصر a ينشئ روابط للانتقال بين الصفحات.
  • التقسيمات (Divisions): عنصر div يستخدم كحاوية لتجميع العناصر معاً.
مثال تطبيقي: الهيكل الأساسي 
<!DOCTYPE html>
<html>
<head>
    <title>تطبيق الويب الأول</title>
</head>
<body>
    <h1>مرحبا بك في تطوير التطبيقات</h1>
    <p>هذه فقرة نصية بسيطة داخل صفحة الويب.</p>
</body>
</html>

التصميم والتنسيق (CSS)

لغة أوراق الأنماط المتتالية (Cascading Style Sheets) هي المسؤولة عن المظهر الخارجي للصفحة. تستطيع من خلالها التحكم بالألوان، الخطوط، التباعد، ومواضع العناصر.

الخواص الشائعة

  • اللون (color): تحديد لون النص.
  • الخلفية (background-color): تحديد لون أو صورة خلفية العنصر.
  • الهوامش (margin & padding): التحكم في المسافات داخل وخارج إطار العنصر.
مثال تطبيقي: تطبيق الاستايل 
body {
    background-color: #f4f4f9;
    font-family: Arial, sans-serif;
}

h1 {
    color: #002855;
    text-align: center;
}

التفاعل مع المستخدم (JavaScript)

تستخدم لغة البرمجة JavaScript لجعل صفحات الويب ديناميكية وتفاعلية. يمكنها تغيير محتوى HTML أو أنماط CSS بناءً على أفعال المستخدم، مثل النقر على زر أو تعبئة نموذج.

مفهوم DOM

يمثل نموذج كائن المستند (Document Object Model) بنية صفحة الويب على شكل شجرة. يمكن لـ JavaScript قراءة هذه الشجرة والتعديل عليها في الوقت الفعلي.

مثال تطبيقي: تغيير محتوى عنصر 
// التأشير على عنصر وتغيير النص الموجود بداخله
let titleElement = document.querySelector('h1');
titleElement.textContent = "تم تغيير النص بواسطة جافا سكربت";

التصميم المتجاوب (Responsive Design)

نظراً لتعدد أحجام الشاشات التي يستخدمها الأشخاص (حواسيب مكتبية، أجهزة لوحية، هواتف ذكية)، أصبح التصميم المتجاوب مطلباً أساسياً.

  • استعلامات الوسائط (Media Queries): تستخدم في CSS لتطبيق أنماط مختلفة بناءً على عرض شاشة الجهاز.
  • الشبكات المرنة (Flexbox & Grid): تقنيات تتيح ترتيب العناصر بشكل مرن يملأ المساحات المتاحة بذكاء.
