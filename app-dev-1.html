فكرة المادة

تطوير التطبيقات 1 يركز على بناء صفحات الويب الأساسية باستخدام HTML وCSS. في هذه المرحلة الهدف أن يفهم الطالب كيف يبني الهيكل الصحيح للصفحة، ثم ينسقه بطريقة واضحة وجميلة ومتجاوبة.

HTML
تبني محتوى الصفحة ومعناها: عنوان، فقرة، رابط، صورة، نموذج، جدول.
CSS
تحدد الشكل: الألوان، الخطوط، المسافات، الحدود، التخطيط، والاستجابة للشاشات.
المتصفح
يقرأ ملفات HTML وCSS ثم يعرض النتيجة للطالب أو المستخدم.
Responsive Design
يجعل الصفحة مناسبة للجوال والتابلت والكمبيوتر.

ما هو تطبيق الويب؟

تطبيق الويب هو برنامج يعمل من خلال المتصفح مثل نظام الجامعة أو صفحة المقرر أو متجر إلكتروني. في هذه المادة ننظر إلى الجزء المرئي من التطبيق، أي الصفحة التي يراها المستخدم ويتعامل معها.

تعريف: صفحة الويب هي مستند HTML منسق بملف CSS، يعرض محتوى منظما للمستخدم داخل المتصفح.
تقريب المفهوم: HTML مثل هيكل المبنى والغرف، وCSS مثل الطلاء والإضاءة وترتيب الأثاث. لا يمكن الاعتماد على الشكل وحده دون هيكل، ولا يظهر الهيكل بشكل جميل دون تنسيق.

HTML: بناء هيكل الصفحة

HTML هي لغة ترميز تصف مكونات الصفحة. كل جزء في الصفحة يكتب غالبا داخل وسم Tag. اختيار الوسم المناسب مهم لأنه يعطي معنى للمحتوى ويساعد المتصفح ومحركات البحث وقارئات الشاشة.

  • Element: عنصر في الصفحة مثل p أو button.
  • Tag: الوسم المستخدم لكتابة العنصر مثل <h1>.
  • Attribute: خاصية تضيف معلومات للعنصر مثل href أو class.
  • Semantic HTML: استخدام عناصر ذات معنى مثل header وmain وsection.

مثال HTML شامل

<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
  <meta charset="UTF-8">
  <title>بطاقة طالب</title>
</head>
<body>
  <header>
    <h1>نظام الطلاب</h1>
    <nav>
      <a href="index.html">الرئيسية</a>
    </nav>
  </header>

  <main>
    <article class="student-card">
      <img src="student.png" alt="صورة الطالب">
      <h2>أحمد محمد</h2>
      <p>التخصص: علوم الحاسب</p>
      <ul>
        <li>HTML</li>
        <li>CSS</li>
      </ul>
      <button>عرض البيانات</button>
    </article>

    <form>
      <label for="name">اسم الطالب</label>
      <input id="name" type="text" placeholder="اكتب الاسم">

      <label for="major">التخصص</label>
      <select id="major">
        <option>علوم الحاسب</option>
        <option>نظم المعلومات</option>
      </select>
    </form>
  </main>

  <footer>كلية الحاسبات</footer>
</body>
</html>

CSS: تنسيق الصفحة

CSS مسؤولة عن الشكل. نستخدمها لتغيير لون النص، الخلفية، حجم الخط، المسافات، الحدود، طريقة توزيع العناصر، والاستجابة لأحجام الشاشات.

Box Model: كل عنصر له محتوى، ثم حشو داخلي padding، ثم حد border، ثم هامش خارجي margin.

التصميم المتجاوب

التصميم المتجاوب يعني أن الصفحة تعمل بشكل جيد على الجوال والتابلت والكمبيوتر، دون أن يضطر المستخدم إلى التكبير أو السحب أفقيا.

قاعدة عملية: ابدأ بتصميم بسيط يعمل على الجوال، ثم وسعه للشاشات الكبيرة. هذه الطريقة تسمى Mobile First.
.cards {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}

@media (max-width: 700px) {
  .cards {
    grid-template-columns: 1fr;
  }
}

تدريب: صمم صفحة "لوحة مقرراتي" تحتوي على عنوان رئيسي، وثلاث بطاقات لمقررات دراسية، وصورة أو أيقونة لكل بطاقة، وزر داخل البطاقة. استخدم HTML وCSS فقط، واجعل البطاقات في عمود واحد على الجوال وثلاثة أعمدة على الشاشة الكبيرة.
