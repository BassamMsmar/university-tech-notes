# ملخص مادة تطوير التطبيقات 1

## فكرة المادة

تطوير التطبيقات 1 يركز على بناء صفحات الويب الأساسية باستخدام HTML وCSS. في هذه المرحلة الهدف هو إتقان الهيكل والتنسيق أولا: كيف نكتب صفحة صحيحة، وكيف نجعلها واضحة وجميلة ومتجاوبة.

## أهداف التعلم

- فهم وظيفة HTML في بناء محتوى الصفحة.
- معرفة أهم أوسمة HTML واستخدام كل وسم في مكانه الصحيح.
- فهم وظيفة CSS في تنسيق الصفحة.
- معرفة أهم خصائص CSS ومحدداتها.
- بناء بطاقة أو صفحة بسيطة باستخدام HTML وCSS فقط.
- تطبيق التصميم المتجاوب للشاشات الصغيرة والكبيرة.

## 1. ما هو تطبيق الويب؟

تطبيق الويب هو برنامج يعمل داخل المتصفح مثل نظام الجامعة أو صفحة المقرر أو المتجر الإلكتروني. في هذه المادة نهتم بالجزء المرئي من التطبيق: الصفحة التي يراها المستخدم.

**تعريف توضيحي:** صفحة الويب هي مستند HTML منسق بملف CSS، يعرض محتوى منظما للمستخدم داخل المتصفح.

**تقريب المفهوم:** HTML مثل هيكل المبنى والغرف، وCSS مثل الطلاء والإضاءة وترتيب الأثاث.

## 2. HTML: بناء هيكل الصفحة

HTML هي لغة ترميز تصف مكونات الصفحة. كل جزء في الصفحة يكتب غالبا داخل وسم Tag. اختيار الوسم المناسب مهم لأنه يعطي معنى للمحتوى ويساعد المتصفح ومحركات البحث وقارئات الشاشة.

### تعريفات مهمة

- **Element:** عنصر في الصفحة مثل `p` أو `button`.
- **Tag:** الوسم المستخدم لكتابة العنصر مثل `<h1>`.
- **Attribute:** خاصية تضيف معلومات للعنصر مثل `href` أو `class`.
- **Semantic HTML:** استخدام عناصر ذات معنى مثل `header`, `main`, `section`, `article`, `footer`.

### أوسمة HTML مهمة

| الوسم | الاستخدام |
|---|---|
| `h1` إلى `h6` | العناوين من الأهم إلى الأقل أهمية |
| `p` | فقرة نصية |
| `a` | رابط لصفحة أو موقع |
| `img` | إضافة صورة |
| `ul`, `ol`, `li` | قوائم نقطية أو رقمية |
| `table`, `tr`, `th`, `td` | الجداول |
| `form` | نموذج إدخال بيانات |
| `label`, `input` | تسمية وحقل إدخال |
| `select`, `option` | قائمة اختيار |
| `textarea` | حقل نص طويل |
| `button` | زر إجراء |
| `div`, `span` | حاويات عامة للتجميع والتنسيق |
| `header`, `nav`, `main`, `section`, `article`, `footer` | تنظيم الصفحة دلاليا |

### مثال HTML شامل

```html
<!DOCTYPE html>
<html lang="ar" dir="rtl">
<head>
  <meta charset="UTF-8">
  <title>بطاقة طالب</title>
</head>
<body>
  <header>
    <h1>نظام الطلاب</h1>
    <nav>
      <a href="index.html">الرئيسية</a>
    </nav>
  </header>

  <main>
    <article class="student-card">
      <img src="student.png" alt="صورة الطالب">
      <h2>أحمد محمد</h2>
      <p>التخصص: علوم الحاسب</p>
      <ul>
        <li>HTML</li>
        <li>CSS</li>
      </ul>
      <button>عرض البيانات</button>
    </article>

    <form>
      <label for="name">اسم الطالب</label>
      <input id="name" type="text" placeholder="اكتب الاسم">

      <label for="major">التخصص</label>
      <select id="major">
        <option>علوم الحاسب</option>
        <option>نظم المعلومات</option>
      </select>
    </form>
  </main>

  <footer>كلية الحاسبات</footer>
</body>
</html>
```

## 3. CSS: تنسيق الصفحة

CSS مسؤولة عن الشكل: الألوان، الخطوط، المسافات، الحدود، الظلال، أماكن العناصر، وحجمها على الشاشات المختلفة.

### تعريفات مهمة

- **Selector:** يحدد العنصر المطلوب تنسيقه مثل `button` أو `.card`.
- **Property:** الخاصية المراد تغييرها مثل `color`.
- **Value:** قيمة الخاصية مثل `blue`.
- **Box Model:** المحتوى، ثم `padding`، ثم `border`، ثم `margin`.

### خصائص ومحددات CSS مهمة

| الخاصية أو المحدد | الاستخدام |
|---|---|
| `color` | لون النص |
| `background` | الخلفية |
| `font-family` | نوع الخط |
| `font-size` | حجم الخط |
| `text-align` | محاذاة النص |
| `margin` | مسافة خارج العنصر |
| `padding` | مسافة داخل العنصر |
| `border` | حد العنصر |
| `border-radius` | استدارة الزوايا |
| `width`, `max-width` | عرض العنصر وحده الأقصى |
| `display: flex` | ترتيب العناصر في اتجاه واحد |
| `display: grid` | تخطيط شبكي |
| `gap` | المسافة بين العناصر |
| `.class` | اختيار عناصر لها class معين |
| `#id` | اختيار عنصر له id معين |
| `:hover` | تنسيق عند مرور المؤشر |

### مثال CSS شامل

```css
body {
  margin: 0;
  font-family: Arial, sans-serif;
  background: #f5f7fb;
  color: #18212f;
}

.student-card {
  max-width: 360px;
  background: white;
  border: 1px solid #dce3ea;
  border-radius: 8px;
  padding: 16px;
  margin: 20px auto;
}

.student-card img {
  width: 100%;
  border-radius: 8px;
}

.student-card button {
  background: #0f4c5c;
  color: white;
  border: 0;
  padding: 10px 16px;
  border-radius: 8px;
}

.student-card button:hover {
  background: #1b6f6a;
}
```

## 4. التصميم المتجاوب Responsive Design

التصميم المتجاوب يعني أن الصفحة تعمل بشكل جيد على الجوال والتابلت والكمبيوتر، دون أن يضطر المستخدم إلى التكبير أو السحب أفقيا.

```css
.cards {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}

@media (max-width: 700px) {
  .cards {
    grid-template-columns: 1fr;
  }
}
```

## مثال تطبيقي صغير

```html
<article class="student-card">

  <h2>أحمد محمد</h2>
  <p>التخصص: علوم الحاسب</p>
  <button>عرض البيانات</button>
</article>
```

صمم صفحة "لوحة مقرراتي" تحتوي على:

- عنوان رئيسي.
- ثلاث بطاقات لمقررات دراسية.
- صورة أو أيقونة لكل بطاقة.
- زر داخل كل بطاقة.
- تنسيق باستخدام CSS فقط.
- عمود واحد على الجوال وثلاثة أعمدة على الشاشة الكبيرة.
